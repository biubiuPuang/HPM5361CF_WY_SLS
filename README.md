# MotorMonitor - 双通道工业通信监控与电机控制

## 项目简介
MotorMonitor 基于 **HPM5361EVK** 开发板，用于监听双电机机器人或工业设备的第一路 RS485/CAN 总线信号。当总线空闲超时或急停按钮触发时，自动切断通信链路，并通过第二路 RS485/CAN 控制两台电机返回原点。期间若重新监听到有效信号，立即恢复链路，确保系统快速响应。

## 功能说明
1. **双通道监听**  
   - 第一路 RS485 或 CAN 进行被动监听（二选一，不能同时使用）。  
   - 只要收到任意有效数据帧（或总线活动），即视为信号有效。
2. **超时阻断**  
   - 若第一路总线持续 **1 秒** 无任何数据，MCU 控制继电器断开外部通信链路，物理上隔离总线。
3. **电机回原点**  
   - 阻断后，第二路 RS485 或 CAN 向两台电机驱动器发送回原点指令，直到电机归零。
4. **信号恢复**  
   - 回原点期间，第一路总线持续监听。一旦检测到有效信号，回原点后并吸合继电器，恢复通信。
5. **急停功能**  
   - 急停按钮按下（低电平有效）时，立刻阻断继电器，执行电机回原点。
6. **系统恢复**  
   - 回到原点后，第一路总线依旧持续监听，一旦检测到有效信号后吸合继电器，恢复通信。


## 硬件接口定义
| 外设 | 引脚/功能 | 备注 |
|------|-----------|------|
| 第一路 RS485/CAN | UART0 (PA00 TX, PA01 RX, PA09 DE) / CAN3 (PB14 RX, PB15 TX) | 监听外部信号总线信息 |
| 第二路 RS485/CAN | UART2 (PB08 TX, PB09 RX, PA10 DE) / CAN0 (PB01 RX, PB00 TX) | 内部控制电机回退 |
| 继电器控制 | PB10 PB11 | 低电平连接外部通信链路，高电连接内部通信链路；PB10 控制RSD485，PB11 控制CAN|
| 备用输出 | PB12 PB13 | 备用两路继电器，低电平有效|
| 急停按钮 | PA12 PA13 | 读取到低电平连通，两个引脚都可选 |
| 调试串口 | UART6 (PA24 TX, PA25 RX) | 用于日志输出和命令行 |
| 外接系统灯 | PA26 | 状态指示灯 低电平有效|
| 内部系统灯 | PA11 | 状态指示灯 低电平有效|
